## Credit
This is my first app with ELM, so far I love it !!
The Socket.IO implementation is heavily inspired from `https://github.com/matthewsj/elm-socketio`

### Installation

* Install elm : `https://guide.elm-lang.org/install.html`
* `elm install`
* `npm install`

### dev
```
$ npm run dev
```

### production
```
$ npm run build
```
